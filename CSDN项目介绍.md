# 🧮 开源方程求解器 - 让数学计算变得优雅而精确

> **一键求解二次、三次、四次方程，自动生成带 LaTeX 公式的详细解题步骤**

## 📌 项目简介

在学习高等数学、工程计算或科研工作中，求解高次方程是经常遇到的难题。市面上的计算器往往只给出数值结果，**缺乏详细的求解过程**，让人知其然而不知其所以然。

本项目是一个**功能全面的方程求解器**，支持二次、三次、四次方程的精确符号求解，并提供：

- ✅ **详细的分步解决方案** - 每一步都清晰可见
- ✅ **LaTeX 数学公式** - 自动生成精美的数学表达式
- ✅ **Markdown 导出** - 方便整理到笔记或论文中
- ✅ **解的自动验证** - 确保结果准确无误
- ✅ **Python + Java 双版本** - 满足不同开发需求

## 🎯 核心功能

### 1️⃣ 支持多种方程类型

| 方程类型 | 数学形式 | 求解方法 |
|---------|---------|---------|
| **二次方程** | $ax^2 + bx + c = 0$ | 求根公式 |
| **三次方程** | $ax^3 + bx^2 + cx + d = 0$ | 卡尔达诺公式 |
| **四次方程** | $ax^4 + bx^3 + cx^2 + dx + e = 0$ | 费拉里方法 |

### 2️⃣ 分步求解展示

以方程 $x^2 - 5x + 6 = 0$ 为例：

```
======================================================================
二次方程求解：ax² + bx + c = 0
======================================================================

步骤 1: 写出标准形式的二次方程
  1x² + -5x + 6 = 0

步骤 2: 确认系数
  a = 1
  b = -5
  c = 6

步骤 3: 计算判别式 Δ = b² - 4ac
  Δ = (-5)² - 4×(1)×(6)
  Δ = 25 - 24
  Δ = 1

步骤 4: 根据判别式判断根的情况
  Δ = 1 方程有两个不相等的实根

步骤 5: 应用求根公式
  求根公式：x = (-b ± √Δ) / (2a)
  代入系数：
  x = (-(-5) ± √(1)) / (2×1)
  x = (5 ± √(1)) / (2)

步骤 6: 计算最终结果
  x₁ = 3
  x₂ = 2
======================================================================
```

## 🎓 数学原理

### 二次方程 - 求根公式

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

判别式 $\Delta = b^2 - 4ac$ 决定根的性质：
- $\Delta > 0$：两个不等实根
- $\Delta = 0$：两个相等实根
- $\Delta < 0$：一对共轭复根

### 三次方程 - 卡尔达诺公式

通过变量代换 $x = t - \frac{b}{3a}$ 将一般三次方程化为缺项形式：

$$t^3 + pt + q = 0$$

然后使用卡尔达诺公式求解。

### 四次方程 - 费拉里方法

通过引入辅助变量，将四次方程转化为求解一个三次预解方程和两个二次方程。
