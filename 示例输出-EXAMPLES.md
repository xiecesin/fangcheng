### 四次方程求解

```
四次方程求解器
================================================
求解：1.0x⁴ - 5.0x² + 4.0 = 0

步骤1: 归一化为首一形式 (a=1)
  已归一化：x⁴ - 5.0x² + 4.0 = 0

步骤2: 四次方程降次 (消除x³项)
  已降次（无x³项）：x⁴ - 5.0x² + 4.0 = 0
  其中 p = -5.0, q = 0.0, r = 4.0

步骤3: 求解预解三次方程: z³ + 2pz² + (p²-4r)z - q² = 0
  z³ - 10.0z² + 9.0z = 0

  求解预解三次方程：
    z = 0.0, 1.0, 9.0  (找到一个正根: z1 = 1.0)

步骤4: 计算 u = √z1 = √1.0 = 1.0

步骤5: 形成两个二次因子：
  因子1: x² + ux + (p + z1 + q/u)/2
         = x² + 1.0x + (-1.0)
         = x² + x - 1.0 = 0

  因子2: x² - ux + (p + z1 - q/u)/2
         = x² - 1.0x + (-1.0)
         = x² - x - 1.0 = 0

步骤6: 求解两个二次方程
  从 x² + x - 1.0 = 0: x = -1.61803, 0.61803
  从 x² - x - 1.0 = 0: x = 1.61803, -0.61803

解：
  x1 = 1.61803
  x2 = -0.61803
  x3 = -1.61803
  x4 = 0.61803

验证：
  验证 x1 = 1.61803: ✓ 误差 = 8.9e-16
  验证 x2 = -0.61803: ✓ 误差 = 1.1e-15
  验证 x3 = -1.61803: ✓ 误差 = 8.9e-16
  验证 x4 = 0.61803: ✓ 误差 = 1.1e-15

所有4个解验证成功！
================================================
```

## 关键特性展示

1. **详细的数学步骤**: 每个求解过程都展示完整的数学推导
2. **精确表达式**: 不仅给出数值解，还显示包含根号的精确形式
3. **验证功能**: 自动验证每个根，显示代入原方程的计算过程
4. **误差分析**: 显示每个根的验证误差值
5. **多种根类型**: 支持实根、复根、重根等多种情况
