## 📝 核心算法实现

### 二次方程:
- 标准求根公式: `x = (-b ± √Δ)/(2a)`
- Δ = b² - 4ac

### 三次方程:
1. 归一化: 除以a
2. 降次: `x = y - b/(3a)`
3. 卡尔达诺公式:
   - 计算: `p = (3ac - b²)/(3a²)`, `q = (2b³ - 9abc + 27a²d)/(27a³)`
   - 求解: `y = ∛(-q/2 + √Δ) + ∛(-q/2 - √Δ)`

### 四次方程:
1. 归一化和降次
2. 解预解三次方程
3. 分解为两个二次方程
4. 分别求解
